# Atabey — AI Governance & Multi-Agent Orchestration Framework (Core Engine)

[![Version](https://img.shields.io/badge/Version-v0.0.17-blue.svg)](https://github.com/ysf-bkr/atabey)
[![npm](https://img.shields.io/npm/v/atabey)](https://www.npmjs.com/package/atabey)
[![License: AGPL-3.0](https://img.shields.io/badge/License-AGPL--3.0-blue.svg)](https://www.gnu.org/licenses/agpl-3.0)

**Atabey** is the core engine of the Agent Atabey framework — an MCP (Model Context Protocol) server designed for AI governance and multi-agent orchestration in developer workflows.

## Installation

```bash
npm install -g atabey
```

## Quick Start

```bash
# Initialize in your project
npx atabey init gemini --profile freelancer --yes

# Verify
npx atabey status
```
